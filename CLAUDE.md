# gstack Engineering Process

## gstack Skills
Always use these specialized skills before building or shipping:
- **Planning:** `/plan-ceo-review`, `/plan-eng-review`, `/plan-design-review`, `/autoplan`
- **Development:** `/office-hours`, `/design-consultation`, `/investigate`, `/codex`, `/cso`
- **Review & Quality:** `/review`, `/design-review`, `/canary`, `/benchmark`, `/qa`, `/qa-only`, `/careful`
- **Operations:** `/ship`, `/land-and-deploy`, `/setup-deploy`, `/setup-browser-cookies`, `/document-release`
- **System:** `/browse`, `/retro`, `/freeze`, `/guard`, `/unfreeze`, `/gstack-upgrade`

## Technical Constraints
- Use the `/browse` skill from gstack for all web browsing.
- Never use `mcp__claude-in-chrome` tools.
